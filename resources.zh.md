# 资源

- [Gnim](https://aylur.github.io/gnim/)：AGS 所基于的基础 GJS 库。注意 `gnim` 模块会从 `ags` 命名空间重新导出。例如 `gnim/gobject` 模块变为 `ags/gobject`。

## Astal 库

- [Apps](https://aylur.github.io/libastal/apps)：用于查询应用程序的库和 CLI 工具
- [Auth](https://aylur.github.io/libastal/auth)：使用 PAM 的身份验证库
- [Battery](https://aylur.github.io/libastal/battery)：upower 守护进程的 DBus 代理库
- [Bluetooth](https://aylur.github.io/libastal/bluetooth)：通过 dbus 控制 bluez 的库
- [Cava](https://aylur.github.io/libastal/cava)：使用 cava 的音频可视化库
- [Greet](https://aylur.github.io/libastal/greet)：向 greetd 发送请求的库和 CLI 工具
- [Hyprland](https://aylur.github.io/libastal/hyprland)：Hyprland IPC 套接字的库和 CLI 工具
- [Mpris](https://aylur.github.io/libastal/mpris)：控制媒体播放器的库和 CLI 工具
- [Network](https://aylur.github.io/libastal/network)：NetworkManager 封装库
- [Notifd](https://aylur.github.io/libastal/notifd)：通知守护进程的库和 CLI 工具
- [PowerProfiles](https://aylur.github.io/libastal/powerprofiles)：控制 upowerd 电源配置文件的库和 CLI 工具
- [River](https://aylur.github.io/libastal/river)：获取 river wayland 合成器状态信息的库和 CLI 工具
- [Tray](https://aylur.github.io/libastal/tray)：系统托盘的库和 CLI 工具
- [WirePlumber](https://aylur.github.io/libastal/wireplumber)：使用 wireplumber 进行音频控制的库
