使用 JavaScript / TypeScript 和 JSX 编写你的桌面 Shell

🚀 初始化项目

使用 ags init 一键生成项目模板，快速上手

📝 TypeScript 类型

使用 ags types 从 GObject 库自动生成类型定义

📦 打包项目

使用 ags bundle 将项目打包成单文件可执行脚本

▶️ 运行项目

使用 ags run 无需预先打包即可直接运行

核心特性

使用熟悉的主流语言

AGS 使用 JavaScript / TypeScript。JSX 语法借鉴 React / Solid 等 Web 框架，让你在 JS 中编写类似 XML 的 UI。

function Bar() {
  const [counter, setCounter] = createState(0)
  const date = createPoll("", 1000, `date "+%H:%M - %A %e."`)

  return (
    <window visible anchor={TOP | LEFT | RIGHT}>
      <centerbox>
        <label $type="start" label={date} />
        <button $type="end" onClicked={() => setCounter((c) => c + 1)}>
          <label label={counter((c) => `已点击 ${c} 次`)} />
        </button>
      </centerbox>
    </window>
  )
}
function BatteryLabel() {
  const pct = createBinding(Battery.get_default(), "percentage")
  return <label label={pct((p) => `${Math.round(p * 100)}%`)} />
}

function MediaPlayers() {
  const players = createBinding(Mpris.get_default(), "players")
  return (
    <For each={players}>
      {(player) => (
        <button
          label={createBinding(player, "title")}
          onClicked={() => player.play_pause()}
        />
      )}
    </For>
  )
}

电池功能已内置

通过 Astal 库直接查询电池、媒体播放、网络等系统信息。后端代码已就绪，你只需专注于构建 UI。

使用 CSS 定制样式

GTK 支持 CSS 样式，AGS 还提供 SASS 支持。涵盖 CSS 变量、关键帧动画、变换等常用特性。
