资源

  • Gnim：AGS 所基于的基础 GJS 库。注意 gnim 模块会从 ags 命名空间重新导出。例如 gnim/gobject 模块变为 ags/gobject。

GJS

  • gjs-docs.gnome.org：为 GJS 注释的库参考文档
  • gjs.guide：GJS 和 GObject 指南

Astal 库

  • Apps：用于查询应用程序的库和 CLI 工具
  • Auth：使用 PAM 的身份验证库
  • Battery：upower 守护进程的 DBus 代理库
  • Bluetooth：通过 dbus 控制 bluez 的库
  • Cava：使用 cava 的音频可视化库
  • Greet：向 greetd 发送请求的库和 CLI 工具
  • Hyprland：Hyprland IPC 套接字的库和 CLI 工具
  • Mpris：控制媒体播放器的库和 CLI 工具
  • Network：NetworkManager 封装库
  • Notifd：通知守护进程的库和 CLI 工具
  • PowerProfiles：控制 upowerd 电源配置文件的库和 CLI 工具
  • River：获取 river wayland 合成器状态信息的库和 CLI 工具
  • Tray：系统托盘的库和 CLI 工具
  • WirePlumber：使用 wireplumber 进行音频控制的库
